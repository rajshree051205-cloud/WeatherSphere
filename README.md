# 🌦️ WeatherSphere – Advanced Real-Time Weather App

A modern and interactive weather application that provides real-time weather updates, dynamic UI effects, live city suggestions, and detailed atmospheric insights for cities around the world.

---

# 🚀 Features

## 🌍 Global Weather Search

* Search weather information for any city worldwide.
* Smart live search suggestions while typing city names.
* Fast API-based real-time data fetching.

## 🎨 Dynamic Weather-Based Backgrounds

The application changes its entire background dynamically according to weather conditions:

* ☀️ Sunny → Bright clear background
* 🌧️ Rainy → Rain-themed background
* ☁️ Cloudy → Cloud atmosphere
* 🌙 Night → Dark aesthetic background
* ❄️ Snow → Snowy visual theme

This creates a realistic and immersive weather experience.

## 📊 Detailed Weather Information

Users can view:

* Temperature
* Humidity
* Pressure
* Wind Speed
* Weather Conditions
* Real-Time Forecast Data

## ⚡ Smooth User Experience

* Beautiful loading spinner while API data is fetching
* Smooth transitions and animations
* All weather cards appear with delayed animation for premium UI feel
* Responsive modern layout

## 🔍 Smart Search Experience

* Instant city suggestions while typing
* Future enhancement planned for recently searched cities
* Easy and clean user interaction

## 📱 Responsive Design

Fully responsive and optimized for:

* Desktop
* Laptop
* Tablet
* Mobile Devices

# ⚙️ How It Works

1. User enters a city name.
2. App fetches real-time weather data using API.
3. Matching city suggestions appear instantly.
4. Weather information is displayed dynamically.
5. Background changes according to weather condition.
6. Smooth animations improve the user experience.
